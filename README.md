# gitchat-nodeSpider
gitchat讲解node爬虫简单demo
本次demo分为三个版本,通过简单的js代码让大家学会node爬虫原理,代码是次要的,只要明白逻辑原理,可以写出很多种代码,这里我仅列举三种方法.


如有问题,请在issue里面提出,欢迎交流,爬虫之路甚远,仍需努力!

### 运行环境
运行环境:nodejs 8.7

如想使用,请下载代码,解压
 执行 npm install
 
 执行 node app.js
